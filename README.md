# SunSaltyBoard

A minimalist floating clipboard manager for Windows.

![SunSaltyBoard](./src-tauri/icons/128x128@2x%20.png)

## Features

- Global hotkey (Ctrl+Shift+V) to show window near cursor
- Click to paste clipboard items
- Search through clipboard history
- Minimalist floating panel design

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```
